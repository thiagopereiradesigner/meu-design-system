# 🚀 Como Importar no Cursor - Guia Rápido

---

## 📁 Passo 1: Organizar Arquivos no Mac

### **No Terminal:**

```bash
# 1. Criar pasta do projeto
mkdir -p ~/Desktop/portal-empresa-ds

# 2. Criar subpastas
mkdir -p ~/Desktop/portal-empresa-ds/src/components
mkdir -p ~/Desktop/portal-empresa-ds/docs
mkdir -p ~/Desktop/portal-empresa-ds/demos
```

### **No Finder:**

```
1. Abrir Downloads (onde estão os arquivos)
2. Abrir Desktop > portal-empresa-ds

Arrastar:
📄 Arquivos .jsx → src/components/
📄 Arquivos .md → docs/
📄 Arquivos .html → demos/
```

## 🎯 Passo 2: Abrir no Cursor

```
1. Abrir Cursor
2. File > Open Folder (ou CMD+O)
3. Navegar até Desktop > portal-empresa-ds
4. Clicar "Open"

✅ Pronto! Cursor agora vê tudo!
```

```

CMD+P         = Buscar arquivo
CMD+SHIFT+F   = Buscar em tudo
CMD+B         = Toggle sidebar
```
