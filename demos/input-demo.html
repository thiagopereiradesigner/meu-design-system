Input

Componente Input | Estados, tokens e código de uso

Campo de texto de linha única com suporte a label, helper text, ícones e estados (normal, erro e disabled).

♿ Melhorias de Acessibilidade Aplicadas

  • Use label textual para contexto do que o usuário deve inserir (além do placeholder).
  • Para erro, aplique .error no container e helper para manter contraste e leitura consistente.
  • Para disabled, use disabled no input e mantenha helper textual.

Anatomia do Componente (HTML)
  • div.field: wrapper do componente.
  • div.field-label: container da label (texto + opcional field-sublabel).
  • span.field-label-text: label principal; com erro use .error (quando aplicável).
  • div.field-input-container: container do input; use .error ou .disabled.
  • input.field-input: campo nativo para foco/teclado.
  • div.field-helper: helper text; com erro aplique .error.

Como Usar (HTML)

Exemplo mínimo de marcação para o input padrão, reaproveitando a estrutura de label, container e helper text. 

<div class="field">
  <div class="field-label">
    <span class="field-label-text">Input label</span>
    <span class="field-sublabel">Sublabel</span>
  </div>
  <div class="field-input-container">
    <input 
      type="text" 
      class="field-input" 
      placeholder="Your text input here" />
  </div>
  <div class="field-helper">
    <span>Helper text opcional</span>
  </div>
</div>
label
Texto exibido acima do campo (classe `.field-label-text`).
sublabel
Texto auxiliar curto ao lado da label (classe `.field-sublabel`).
helper text
Mensagem explicativa ou de erro logo abaixo do campo (classe `.field-helper`).
states
Use as classes .error em .field-input-container e .field-helper para estado de erro, e .disabled + disabled no input para estado desabilitado.
Como Usar (React)

O snippet abaixo replica a mesma estrutura de classes do HTML, usando estado local para o valor do <input>. 

import React, { useState } from 'react';

function InputField({
  label,
  sublabel,
  placeholder = 'Your text input here',
  helperText = 'Helper text opcional',
  error = false,
  disabled = false,
}) {
  const [value, setValue] = useState('');
  const isError = Boolean(error);

  return (
    <div className="field">
      <div className="field-label">
        <span className="field-label-text">{label}</span>
        {sublabel ? <span className="field-sublabel">{sublabel}</span> : null}
      </div>

      <div className={"field-input-container" + (isError ? " error" : "") + (disabled ? " disabled" : "")}>
        <input
          type="text"
          className="field-input"
          placeholder={placeholder}
          disabled={disabled}
          value={value}
          onChange={(e) => setValue(e.target.value)}
        />
      </div>

      <div className={"field-helper" + (isError ? " error" : "")}>
        <span>{helperText}</span>
      </div>
    </div>
  );
}
label
Texto principal da .field-label-text.
sublabel
Opcional; renderizado como .field-sublabel.
helperText
Conteúdo do bloco .field-helper.
error
Quando true, aplica classes error e foca no estado de erro.
disabled
Quando true, aplica classe disabled e define disabled no input.
