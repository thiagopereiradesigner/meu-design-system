Iconografia

Biblioteca de ícones do design system baseada no Lucide Icons. Clique em qualquer ícone para visualizar detalhes e fazer download em SVG.

Referência: lucide.dev

Use a biblioteca para copiar SVGs compatíveis com `stroke="currentColor"` e reaproveitar a mesma iconografia em telas e componentes.

♿ Melhorias de Acessibilidade Aplicadas

  • Botões do drawer possuem aria-label (ex.: fechar).
  • Fechamento do drawer via teclado com Escape.
  • O SVG exportado mantém fill="none" e stroke="currentColor" para herdar cor e manter consistência visual.
Anatomia do Componente (HTML)
  • div#category-filters: lista de filtros por categoria (renderizada por JS).
  • input#icon-search: busca do grid de ícones.
  • div#icons-container: container onde os cards de ícone são renderizados.
  • .icon-card[data-icon]: cada ícone clicável abre o drawer com preview.
  • div#drawer + div#drawer-overlay: painel lateral com preview, tamanhos e ações.
  • div#drawer-preview: onde o SVG do ícone selecionado é injetado.
  • button#btn-copy-svg e button#btn-download-svg: copiar/download do SVG.
  • div#toast: feedback após copiar/download.

Como Usar (React)

Snippet JSX para renderizar um ícone com cor via currentColor. 

import React from 'react';

export function SearchIcon({
  size = 20,
  className,
  title,
  ariaLabel,
}) {
  return (
    <svg
      width={size}
      height={size}
      viewBox="0 0 24 24"
      fill="none"
      stroke="currentColor"
      strokeWidth="2"
      strokeLinecap="round"
      strokeLinejoin="round"
      className={className}
      role={ariaLabel ? "img" : "presentation"}
      aria-label={ariaLabel}>
      {title ? <title>{title}</title> : null}
      <circle cx="11" cy="11" r="8"/>
      <path d="m21 21-4.3-4.3"/>
    </svg>
  );
}
size
Tamanho do SVG (equivalente ao width/height).
className
Permite ajustar cor via CSS (ex.: color: var(...)).
title
Opcional; adiciona legenda interna para acessibilidade.
ariaLabel
Quando presente, o SVG vira role="img".
