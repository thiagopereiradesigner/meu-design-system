Switch

Alternância ligado/desligado para preferências binárias (notificações, modo escuro, etc.). Três tamanhos (sm / md / lg), estados desabilitados e suporte a teclado com role="switch".

Acessibilidade

  • role="switch" e aria-checked refletem o estado (atualizado pelo script desta página).
  • Teclado: Enter e Espaço alternam quando o trilho está focável (tabindex="0").
  • disabled: sem foco no trilho; aria-disabled="true" e opacidade reduzida.
  • Em viewports estreitas, o rótulo + padding da linha ampliam a área tocável (~44px de altura).
Anatomia do Componente (HTML)
  • label.switch-wrapper (+ opcional switch-wrapper--sm / --lg): alinha trilho e texto; aumenta área clicável.
  • div.switch-track (+ switch-track--sm | switch-track--lg; padrão = médio): trilho; classes checked e disabled.
  • div.switch-thumb: indicador que desliza; posição vem do CSS (calc com variáveis do tamanho).
  • span.switch-label: texto do controle.
Como Usar (HTML)

Três larguras via modificadores no trilho (switch-track--sm, padrão médio, switch-track--lg). Opcional: classes switch-wrapper--sm / --lg para escalar a tipografia do rótulo.

<label class="switch-wrapper switch-wrapper--sm">
  <div class="switch-track switch-track--sm" role="switch" aria-checked="false" tabindex="0">

</label>
Como Usar (React)

O componente Switch usa variáveis CSS do design system (--ds-success, --ds-n200, --ds-font-family, etc.) quando a folha de tokens/tema estiver carregada. 

import { Switch } from '@/components/Forms/Switch/Switch';

<Switch label="Notificações" size="small" checked={on} onChange={setOn} />
<Switch label="Padrão" size="medium" checked={x} onChange={setX} />
<Switch label="Destaque" size="large" checked={y} onChange={setY} />
label
Texto exibido ao lado do trilho.
checked / onChange
Modo controlado; sem checked, o componente mantém estado interno.
disabled
Bloqueia interação e reduz opacidade.
size
"small" | "medium" (padrão) | "large".
labelPosition
"right" (padrão) ou "left".
