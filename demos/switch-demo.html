Componente Switch | On/Off acessível

Switch acessível para alternar estados (on/off), com suporte a disabled e navegação via teclado.

♿ Melhorias de Acessibilidade Aplicadas

  • Use um elemento focável (ou role="switch") para anunciar o estado ao leitor de tela.
  • Quando estiver em disabled, mantenha o switch não interativo e com estado visual coerente.
  • Garanta que o estado atual seja refletido imediatamente na UI (classe .checked).

🎚️ Switch

Componente para alternar rapidamente entre dois estados (ligado/desligado).

Basic

States

Anatomia do Componente (HTML)
  • label.switch-wrapper: rótulo clicável (aumenta área de interação).
  • div.switch-track: trilho do switch; use .checked para ligado e .disabled para desabilitado.
  • div.switch-thumb: “bolinha” que se move quando o switch é ativado.
  • span.switch-label: texto do controle.
  • JS: função toggleSwitch(element) alterna .checked e respeita .disabled.

Como Usar (HTML)

Estrutura mínima de um switch controlado via JavaScript. 

<label class="switch-wrapper">
  <div class="switch-track" onclick="toggleSwitch(this)">
    <div class="switch-thumb"></div>
  </div>
  <span class="switch-label">Ativar notificações</span>
</label>
Como Usar (React)

Snippet React para controlar checked no estado e manter o mesmo layout HTML. 

import React, { useState } from 'react';

function SwitchField({ label, disabled = false, defaultChecked = false }) {
  const [checked, setChecked] = useState(defaultChecked);

  return (
    <label className="switch-wrapper">
      <div
        className={"switch-track" + (checked ? " checked" : "") + (disabled ? " disabled" : "")}
        onClick={() => {
          if (disabled) return;
          setChecked((v) => !v);
        }}
        role="switch"
        aria-checked={checked}
      >
        <div className="switch-thumb"></div>
      </div>
      <span className="switch-label">{label}</span>
    </label>
  );
}
label
Texto do controle (span.switch-label).
disabled
Quando true, bloqueia click e adiciona classe .disabled.
defaultChecked
Estado inicial do switch (equivalente a .checked).
