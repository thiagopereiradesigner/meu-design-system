Radio

O radio permite escolher uma única opção entre várias no mesmo grupo (name igual). Use em formulários de pagamento, preferências ou qualquer fluxo em que as alternativas se excluam mutuamente.

♿ Nota Acessibilidade

  • Prefira input type="radio" nativo com label envolvendo ou associado por for/id; mantenha foco visível (:focus-visible / box-shadow: var(--ds-focus-ring)).
  • Agrupe opções relacionadas com o mesmo name; em React, o RadioGroup usa role="radiogroup" com aria-labelledby no título do grupo e aria-describedby na mensagem de erro ou texto de ajuda.
  • Para disabled, preserve o estado no DOM e evite depender só de opacidade: contraste ainda deve ser legível; não remova o foco de elementos habilitados adjacentes.
  • Com erro de validação, exponha mensagem ligada ao grupo (aria-invalid, texto de erro legível).

Anatomia do Componente (HTML)

Radio group com input nativo e classes para tamanho, estados e erro.

  • label.radio-wrapper: área clicável; use radio-wrapper--error quando o grupo estiver inválido.
  • input[type="radio"].radio-input: controle; modifique com radio-input--sm, radio-input--md, radio-input--lg; atributo disabled para inativo.
  • span.radio-label (e radio-label--sm / radio-label--lg): rótulo da opção.
  • div.helper-text: texto de apoio ou mensagem de erro abaixo do grupo.

Como Usar (React)

Exemplo usando o componente do design system. 

{/* React */}
import React, { useState } from 'react';
import { RadioGroup } from '@/components/Forms/Radio/Radio';

export default function Demo() {
  const [value, setValue] = useState('credit');

  return (
    <RadioGroup
      label="Método de pagamento"
      name="payment"
      value={value}
      onChange={setValue}
      size="md"
      options={[
        { label: 'Cartão de crédito', value: 'credit' },
        { label: 'Cartão de débito', value: 'debit' },
        { label: 'PIX indisponível', value: 'pix', disabled: true },
      ]}
    />
  );
}
Props principais
label, options (label, value, disabled opcional), value, onChange
size ('sm' | 'md' | 'lg'), direction, disabled, error, helperText, name
