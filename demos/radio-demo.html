Anatomia do Componente (HTML)

Radio group com input nativo e classes para estilização (estado checked/disabled).

  • label.radio-wrapper: wrapper clicável.
  • input[type="radio"].radio-input: input nativo (radio).
  • span.radio-label: texto da opção.
  • div.helper-text: texto de apoio/erro no grupo.

Como Usar (HTML)

Estrutura base para um grupo de radios vertical. 

<div>
  <div>Método de pagamento</div>
  <label class="radio-wrapper">
    <input type="radio" class="radio-input" name="payment" value="credit" checked />
    <span class="radio-label">Cartão de crédito</span>
  </label>
  <label class="radio-wrapper">
    <input type="radio" class="radio-input" name="payment" value="debit" />
    <span class="radio-label">Cartão de débito</span>
  </label>
</div>

Como Usar (React)

Exemplo usando o componente do design system. 

{/* React */}
import React, { useState } from 'react';
import { RadioGroup } from '@/components/Forms/Radio/Radio';

export default function Demo() {
  const [value, setValue] = useState('credit');

  return (
    <RadioGroup
      label="Método de pagamento"
      name="payment"
      value={value}
      onChange={setValue}
      options={[
        { label: 'Cartão de crédito', value: 'credit' },
        { label: 'Cartão de débito', value: 'debit' },
      ]}
    />
  );
}
Props principais
label, options, value, onChange
direction (vertical/horizontal), disabled, error, helperText

♿ Nota Acessibilidade

  • Inputs nativos (type="radio") preservam navegação por teclado.
  • Use label e helperText para contexto.
  • Para erro, passe error e garanta contraste.
