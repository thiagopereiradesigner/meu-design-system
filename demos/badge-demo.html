Badge

Variações por tipo (success/error/warning/info/neutral) e um modo opcional de exibir ícone ao lado do texto.

♿ Melhorias de Acessibilidade Aplicadas

  • Uso consistente de contraste: cores do badge vêm das variáveis do DS.
  • Foco e legibilidade: textos e ícones usam tamanho mínimo e stroke visível.
Anatomia do Componente (HTML)
  • span.badge: componente base (pill/arredondado e alinhamento).
  • span.badge.success|error|warning|info|neutral: variação semântica de cor via classes.
  • span.badge.small|medium: controle de tamanho (padding e font-size).
  • span.icon (opcional): ícone à esquerda.
  • span.right-icon (opcional): ícone à direita (ex.: chevron-right).
  • border-radius: sempre em “full” (pill) para consistência.
Como Usar (HTML)
O badge é puramente visual: escolha a variação (color) e decida se vai exibir icon (esquerda) e/ou right-icon (direita). 
{/* HTML */}
<span class="badge success small">
  <!-- sem ícone -->
  Ativo
</span>

<span class="badge error medium">
  <span class="icon">
    <!-- svg do ícone aqui -->
  </span>
  Erro
</span>

<span class="badge info medium">
  Info
  <span class="right-icon">
    <!-- svg chevron-right aqui -->
  </span>
</span>
Com ícone
Use quando o status precisa ser reconhecido mais rápido.

Sem ícone

Como Usar (React)
`Badge` é um componente inline para status/categoria. O formato é sempre pill (radius full) e as cores vêm de CSS variables (white-label). 
        {/* JSX */}
import { Badge } from '@/components/ui/Badge';
