Accordion

Estilo semelhante ao shadcn, incluindo divider após a resposta do item aberto.

♿ Melhorias de Acessibilidade Aplicadas

  • Keyboard friendly: trigger utiliza botão com aria-expanded.
  • Leitura assistida: conteúdo usa role="region".
  • Divider “grudado”: traço aparece após a resposta aberta, mantendo contexto.
Perguntas frequentes
Você consegue montar interfaces consistentes usando tokens e componentes do design system. O objetivo aqui é acelerar o desenvolvimento e manter acessibilidade.
Você altera tokens CSS (cores e tipografia) via variáveis. Os componentes permanecem com a mesma estrutura e semântica.
Sim: o trigger usa `aria-expanded`, e o conteúdo respeita leitura consistente por teclado.
Anatomia do Componente (HTML)
  • div.acc-item: item da lista (estado open/closed via classe open).
  • button.acc-trigger: botão que controla aria-expanded.
  • span.acc-index + span.acc-title: separa o “número” do título (o índice não fica tão pesado).
  • div.acc-content: container colapsável (altura via CSS).
  • div.acc-content-inner: conteúdo de leitura.
  • div.acc-divider: separador exibido quando o item está aberto.
Como Usar (HTML)
Use o data-value para identificar o item e a função toggleAcc para alternar a classe open e o aria-expanded. 
{/* HTML + JS */}
<div class="accordion" id="accordion">
  <div class="acc-item open" data-value="1">
    <button class="acc-trigger" onclick="toggleAcc(1)" aria-expanded="true">
      <span class="acc-text">
        <span class="acc-index">1.</span>
        <span class="acc-title">Pergunta</span>
      </span>
      <svg class="acc-chevron" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
        <path d="M6 9l6 6 6-6" />
      </svg>
    </button>

    <div class="acc-content">
      <div class="acc-content-inner">Resposta...</div>
    </div>

    <div class="acc-divider"></div>
  </div>
</div>

<script>
  function toggleAcc(val) {
    const items = document.querySelectorAll(".acc-item");
    items.forEach((el) => {
      const isOpen = String(el.getAttribute("data-value")) === String(val);
      el.classList.toggle("open", isOpen);
      const btn = el.querySelector(".acc-trigger");
      if (btn) btn.setAttribute("aria-expanded", String(isOpen));
      const chevron = el.querySelector(".acc-chevron");
      if (chevron) chevron.style.transform = isOpen ? "rotate(180deg)" : "rotate(0deg)";
    });
  }
</script>
Como Usar (React)
Este Accordion renderiza o conteúdo de cada item e, quando aberto, exibe um `Divider` logo após a resposta (estilo shadcn). Para o número não ficar com o mesmo peso do título, o trigger separa “1.” do texto. 
      {/* JSX */}
import {
  Accordion,
  AccordionItem,
  AccordionTrigger,
  AccordionContent,
} from '@/components/ui/Accordion';


  
    1. O que você precisa saber?
    
      Texto da resposta...
Props/observações
AccordionContent: withDivider controla o traço após a resposta
AccordionTrigger: neste demo, o formato do texto “1. ...” é usado para exibir índice + título com pesos diferentes
