Tooltip (com arrow)

Dica contextual exibida ao passar o mouse ou focar o elemento (com setinha).

♿ Melhorias de Acessibilidade Aplicadas

  • Tooltip usa role="tooltip" e aria-describedby no trigger.
  • Suporta teclado: exibe também em focus/blur.
  • Delay controlado no JS para reduzir poluição visual.

Anatomia do Componente (HTML)
  • div.tooltip-container: wrapper do trigger + tooltip.
  • button (ou trigger): elemento alvo com aria-describedby.
  • div.tooltip: conteúdo do tooltip (setinha via ::after).
  • role="tooltip": identifica a região como tooltip.
Como Usar (HTML)
Use um wrapper e um par trigger + tooltip com id/aria-describedby.
{/* HTML + JS mínimo */}
<div class="tooltip-container" data-delay="200">
  <button type="button" aria-describedby="tip-top">Top</button>
  <div id="tip-top" class="tooltip tooltip-top" role="tooltip">
    Tooltip no topo
  </div>
</div>

<script>
  (function setupTooltips(){
    document.querySelectorAll('.tooltip-container').forEach((container) => {
      const btn = container.querySelector('button');
      const tooltip = container.querySelector('.tooltip');
      const delay = Number(container.getAttribute('data-delay') || 200);

      let t = null;
      function show(){ tooltip.classList.add('show'); }
      function hide(){ tooltip.classList.remove('show'); }

      btn.addEventListener('mouseenter', () => { clearTimeout(t); t = setTimeout(show, delay); });
      btn.addEventListener('mouseleave', () => { clearTimeout(t); hide(); });
      btn.addEventListener('focus', () => { clearTimeout(t); t = setTimeout(show, delay); });
      btn.addEventListener('blur', () => { clearTimeout(t); hide(); });
    });
  })();
</script>
Como Usar (React)
A API do componente segue: content, position e delay.
{/* JSX */}
import { Tooltip } from '@/components/FinalComponents';

function Example() {
  return (
    <Tooltip content="Tooltip no topo" position="top" delay={200}>
      <button type="button">Top</button>
    </Tooltip>
  );
}
Props principais
content: texto da dica.
position: top | right | bottom | left.
delay (opcional): ms para exibir.
