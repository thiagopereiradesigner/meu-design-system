Stepper

Verde = steps passados e o step atual. Cinza = o que falta. O nome do step atual fica abaixo.

♿ Melhorias de Acessibilidade Aplicadas

  • Sticky consistente: stepper permanece visível durante o fluxo.
  • Contraste: cores baseadas no design system (sucesso/borda).
  • Rótulo do passo atual: ajuda navegação e entendimento do progresso.
Anatomia do Componente (HTML)
  • div.stepper: contêiner do progresso (wrapper).
  • div.step-row: linha onde os segmentos div.bar-segment são renderizados.
  • div.bar-segment.active: segmento verde (passados + step atual).
  • div.step-label: rótulo do step atual (texto abaixo do progresso).
Neste demo, o markup do row/label fica fixo e os segmentos são adicionados via JS para facilitar a reutilização.
Como Usar (HTML)
Copie o contêiner + IDs e use o JS equivalente para gerar os bar-segment com base em currentStep. 
{/* HTML + JS */}
<div class="stepper" id="stepper">
  <div class="step-row" id="step-row"></div>
  <div class="step-label" id="step-label"></div>
</div>

<script>
  const steps = [
    { label: 'Termos' },
    { label: 'Confirmação com PIN' },
    { label: 'Sucesso' },
  ];

  let currentStep = 1;

  function renderStepper() {
    const stepRow = document.getElementById('step-row');
    const stepLabel = document.getElementById('step-label');
    stepRow.innerHTML = '';

    steps.forEach((s, idx) => {
      const seg = document.createElement('div');
      seg.className = 'bar-segment';
      if (idx <= currentStep) seg.classList.add('active');
      stepRow.appendChild(seg);
    });

    stepLabel.textContent = steps[currentStep]?.label || '';
  }

  function setStep(i) {
    currentStep = Math.max(0, Math.min(i, steps.length - 1));
    renderStepper();
  }

  renderStepper();
</script>

Como Usar (React)
O título do step atual deve ficar centralizado e com fonte reduzida para não “estourar” o fluxo. O stepper é sticky logo abaixo do header.
Exemplo
{/* JSX */}
import { Stepper } from '@/components/ui/Stepper';
Props principais
  • steps: lista de labels dos passos
  • currentStep: índice do step atual (0-based)
  • sticky (opcional): mantém o stepper fixo abaixo do header
  • offsetTop (opcional): altura do header (default: 56)
