Stepper

Barra de progresso por segmentos: o passo atual e os anteriores usam a cor de sucesso; o que falta fica neutro. O rótulo do passo atual aparece abaixo da barra. Use em fluxos curtos (ex.: termos → confirmação → sucesso).

Acessibilidade

  • Sticky: o stepper permanece visível ao rolar, alinhado à topbar.
  • Contraste: cores via tokens do DS (--ds-success, --ds-border).
  • Rótulo textual: o passo atual é anunciado visualmente; em produto, associe a aria-current="step" no passo ativo quando for navegação por lista.
  • Teclado: estes botões de demo têm foco visível (:focus-visible).
Anatomia do Componente (HTML)
  • div.stepper: contêiner do progresso (wrapper sticky).
  • div.step-row: linha onde os segmentos div.bar-segment são renderizados.
  • div.bar-segment.active: segmento preenchido (passados + passo atual).
  • div.step-label: rótulo do passo atual (texto abaixo da barra).
Neste demo, o markup do row e do label fica fixo e os segmentos são adicionados via JS para facilitar a reutilização.
Como Usar (HTML)
Copie o contêiner + IDs e use o JS equivalente para gerar os bar-segment com base em currentStep. 
<div class="stepper" id="stepper">
  <div class="step-row" id="step-row"></div>
  <div class="step-label" id="step-label"></div>
</div>

<script>
  const steps = [
    { label: 'Termos' },
    { label: 'Confirmação com PIN' },
    { label: 'Sucesso' },
  ];

  let currentStep = 1;

  function renderStepper() {
    const stepRow = document.getElementById('step-row');
    const stepLabel = document.getElementById('step-label');
    stepRow.innerHTML = '';

    steps.forEach((s, idx) => {
      const seg = document.createElement('div');
      seg.className = 'bar-segment';
      if (idx <= currentStep) seg.classList.add('active');
      stepRow.appendChild(seg);
    });

    stepLabel.textContent = steps[currentStep]?.label || '';
  }

  function setStep(i) {
    currentStep = Math.max(0, Math.min(i, steps.length - 1));
    renderStepper();
  }

  renderStepper();
</script>

Como Usar (React)
O título do passo atual deve ficar centralizado e com tipografia reduzida para não competir com o conteúdo do fluxo. O stepper pode ser sticky logo abaixo do header; use o token CSS --ds-topbar-height (ou a prop offsetTop) para alinhar a outros temas.
Exemplo
import { Stepper } from '@/components/ui/Stepper';

<Stepper
  steps={[
    { label: 'Termos' },
    { label: 'Confirmação com PIN' },
    { label: 'Sucesso' },
  ]}
  currentStep={1}
/>
Props principais
  • steps: lista de labels dos passos
  • currentStep: índice do passo atual (0-based)
  • sticky (opcional): mantém o stepper fixo abaixo do header
  • offsetTop (opcional): deslocamento em px (equivalente a --ds-topbar-height no HTML/CSS)
