Drawer

Painel lateral que abre sobre a tela e fecha por overlay, botão e tecla Esc.

♿ Melhorias de Acessibilidade Aplicadas

  • Drawer usa role="dialog" e aria-modal="true".
  • Fechamento por overlay (clique), botão e Esc.
  • Ao abrir, o foco vai para o botão de fechar.
  • Campo de texto: container próprio; Categoria reutiliza o Select custom de select-demo.html (ds-demo-base.css + ds-demo-select.js).
  • Labels com --ds-text-primary; ícone de fechar com aria-label e foco visível.
  • Respeita prefers-reduced-motion e env(safe-area-inset-*) em mobile.

Anatomia do Componente (HTML)
  • button de abrir (openDrawer()).
  • div.overlay#drawer-overlay: sombra e clique externo para fechar.
  • div.drawer#drawer: container do dialog.
  • role="dialog" + aria-modal="true": acessibilidade.
  • div.drawer-header + button.close-btn + div.drawer-content.
  • Formulário: .drawer-field + texto em .drawer-field-control; categorias com .select-container + .field-input-container + .select-dropdown (igual à página Select).
  • Inferior flutuante: .drawer--bottom (largura máx. centrada em desktop). Inferior ponta a ponta: .drawer--bottom.drawer--bottom-full.
Como Usar (HTML)
Estruture overlay + drawer e use openDrawer()/closeDrawer().
{/* HTML + JS mínimo */}
<button id="drawer-open-btn" type="button" onclick="openDrawer()">
  Abrir Drawer
</button>

<div class="overlay" id="drawer-overlay" onclick="closeDrawer()"></div>

<div class="drawer" id="drawer" role="dialog" aria-modal="true" aria-labelledby="drawer-title">
  <div class="drawer-header">
    <h2 class="drawer-title" id="drawer-title">Configurações</h2>
    <button class="close-btn" type="button" onclick="closeDrawer()" aria-label="Fechar">...</button>
  </div>
  <div class="drawer-content">...</div>
</div>

<script>
  function openDrawer(){
    document.getElementById('drawer-overlay').classList.add('show');
    document.getElementById('drawer').classList.add('show');
    document.body.style.overflow = 'hidden';
    document.getElementById('drawer-close-btn').focus();
  }

  function closeDrawer(){
    document.getElementById('drawer-overlay').classList.remove('show');
    document.getElementById('drawer').classList.remove('show');
    document.body.style.overflow = '';
    document.getElementById('drawer-open-btn').focus();
  }

  document.addEventListener('keydown', (e) => {
    if (e.key === 'Escape') closeDrawer();
  });
</script>
Como Usar (React)
O componente React controla isOpen e fecha via onClose.
{/* JSX */}
import { Drawer } from '@/components/FinalComponents';
import React, { useState } from 'react';

function Page() {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <>
      <button type="button" onClick={() => setIsOpen(true)}>Abrir Drawer</button>
      <Drawer
        isOpen={isOpen}
        onClose={() => setIsOpen(false)}
        title="Configurações"
        position="right"
        width="400px"
      >
        <button type="button">Salvar</button>
      </Drawer>
    </>
  );
}
Props principais
isOpen: controla exibição do drawer.
onClose: callback ao fechar (overlay/esc/botão).
title (opcional): título no header.
position (opcional): left | right.
width (opcional): string (ex.: "400px").
