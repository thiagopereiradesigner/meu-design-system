Breadcrumb

Navegação hierárquica (caminho) para orientar o usuário em páginas aninhadas.

♿ Melhorias de Acessibilidade Aplicadas

  • Uso de nav aria-label="Breadcrumb".
  • Último item usa aria-current="page" (estado atual).
  • Hover/foco: cor de texto var(--ds-success-fg) sobre var(--ds-success-bg) (contraste); foco visível com var(--ds-focus-ring).
  • Mobile: links com altura mínima tocável (44px) em viewports estreitas; desktop compacta o padding.

Anatomia do Componente (HTML)
  • nav.breadcrumb: wrapper semântico da navegação.
  • a: links dos níveis anteriores (hover semibold + fundo success-bg / texto success-fg).
  • span.separator: separador entre níveis (ex.: / ou >).
  • span.current + aria-current="page": item atual.
Como Usar (HTML)
Estruture links + separadores + item atual com aria-current.
{/* HTML */}
<nav class="breadcrumb" aria-label="Breadcrumb">
  <a href="/">Home</a>
  <span class="separator">/</span>
  <a href="/products">Produtos</a>
  <span class="separator">/</span>
  <span class="current" aria-current="page">Notebook Dell</span>
</nav>
Como Usar (React)
Passe items (com href) e ajuste separator quando necessário.
{/* JSX */}
import { Breadcrumb } from '@/components/FinalComponents';

function Example() {
  return (
    <Breadcrumb
      items={[
        { id: 'home', label: 'Home', href: '/' },
        { id: 'products', label: 'Produtos', href: '/products' },
        { id: 'category', label: 'Eletrônicos', href: '/products/electronics' },
        { id: 'current', label: 'Notebook Dell' },
      ]}
      separator="/"
    />
  );
}
Props principais
items: {`{ id, label, href? }[]`}
separator (opcional): string.
