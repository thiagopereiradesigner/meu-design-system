Spinner / Loader

Indicador visual de carregamento (uso em telas e botões durante operações assíncronas).

♿ Melhorias de Acessibilidade Aplicadas

  • role="status" e aria-label comunicam carregamento para leitores de tela.
  • Texto oculto via .sr-only evita “loading sem contexto”.
  • Animação em 0.8s e loop contínuo para manter feedback.

Anatomia do Componente (HTML)
  • div.spinner: spinner (animação + borda).
  • spinner-small|spinner-medium|spinner-large: define o tamanho.
  • role="status": comunica estado (carregando).
  • aria-label="Carregando": texto para leitores de tela.
  • span.sr-only: texto oculto “Carregando...” para contexto adicional.
Como Usar (HTML)
Markup com A11y + classes de tamanho para consistência.
{/* HTML */}
<div class="spinner spinner-medium" role="status" aria-label="Carregando">
  <span class="sr-only">Carregando...</span>
</div>
Como Usar (React)
Use a prop size para ajustar o tamanho e, se necessário, color.
{/* JSX */}
import { Spinner } from '@/components/FinalComponents';

function Loading() {
  return (
    <Spinner size="large" />
  );
}
Props principais
size: 'small' | 'medium' | 'large'
color (opcional): cor do “top border” (string/hex)
