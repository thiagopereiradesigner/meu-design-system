✨ Melhorias Aplicadas

  • Ícones do Lucide: circle-alert e x
  • Título + Mensagem: Opcional via `showText` (com body) e showBody (somente título).
  • Texto alinhado à esquerda: Conforme especificação
  • Animação suave: Fade + Slide from top (40px)
  • Auto-close: 3s (success), 4s (info/warning), 5s (error)
  • Progress bar: Animada, pausa no hover
  • Esc fecha: Acessibilidade de teclado
  • WCAG AA: Contraste mínimo 4.5:1
Anatomia do Componente (HTML)
  • div#toast-container.toast-container: contêiner fixo (onde as toasts são injetadas).
  • div.toast: wrapper da notificação (criado via JS no demo).
  • div.toast-icon: ícone do tipo (Lucide-style).
  • div.toast-content: agrupador do texto quando showText=true.
  • div.toast-title: título (quando habilitado).
  • div.toast-message: body/mensagem (quando habilitado).
  • button.toast-close: fecha a toast; suportar teclado via foco (botão nativo).
  • div.toast-progress + div.toast-progress-bar: barra de progresso por duração.
Como Usar (HTML)
Copie o contêiner e, no JS, chame showToast(type, variant, showText, showBody). Use showText=true + showBody=false para ícone + título (sem body). 
{/* HTML + JS (min) */}
<div id="toast-container" class="toast-container" role="alert" aria-live="polite" aria-atomic="true"></div>

<!-- Depois de incluir/ter as funções showToast/hideToast do demo -->
<script>
  showToast('success', 'active', true);
  showToast('error', 'active', false);
</script>
