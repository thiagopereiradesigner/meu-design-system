Menu / Sidebar

Navegação lateral colapsável com estado ativo (clique ou teclado).

♿ Melhorias de Acessibilidade Aplicadas

  • Itens são button (teclado via Tab e Enter/Space).
  • Estado ativo comunica via aria-current="page".
  • Menu usa <nav aria-label="Menu"> para leitores de tela.
Preview
Anatomia do Componente (HTML)
  • nav.menu: container da sidebar (colapsável via classe collapsed).
  • button.menu-item: item de navegação.
  • button.menu-item.active: item selecionado (estilo e aria-current).
  • button.menu-item span: label (oculta quando .menu.collapsed).
Como Usar (HTML)
Use menu.collapsed para colapsar e menu-item.active para o item atual.
{/* HTML + JS mínimo */}
<button type="button" onclick="toggleMenu()">
  <span id="menu-toggle-text">Colapsar Menu</span>
</button>

<nav class="menu" id="sidebar-menu" aria-label="Menu">
  <button class="menu-item active" type="button" onclick="setActive(this)" aria-current="page">
    <svg>...</svg>
    <span>Dashboard</span>
  </button>
</nav>

<script>
  function toggleMenu() {
    const menu = document.getElementById('sidebar-menu');
    const text = document.getElementById('menu-toggle-text');
    menu.classList.toggle('collapsed');
    text.textContent = menu.classList.contains('collapsed')
      ? 'Expandir Menu'
      : 'Colapsar Menu';
  }

  function setActive(btn) {
    document.querySelectorAll('.menu-item').forEach((el) => {
      el.classList.remove('active');
      el.removeAttribute('aria-current');
    });
    btn.classList.add('active');
    btn.setAttribute('aria-current', 'page');
  }
</script>
Como Usar (React)
Passe items, controle collapsed e opcionalmente trate onItemClick.
{/* JSX */}
import { Menu } from '@/components/FinalComponents';

const menuItems = [
  { id: 'dashboard', label: 'Dashboard', icon: <svg /> },
  { id: 'users', label: 'Usuários', icon: <svg /> },
];

function Sidebar() {
  return (
    <Menu
      items={menuItems}
      collapsed={false}
      onItemClick={(item) => console.log(item.id)}
    />
  );
}
Props principais
items: {`{ id, label, icon? }`}[]
collapsed (opcional): boolean
onItemClick (opcional): (item) => void
