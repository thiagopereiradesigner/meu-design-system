Menu / Sidebar

Navegação lateral colapsável com estado ativo (clique ou teclado).

♿ Melhorias de Acessibilidade Aplicadas

  • Itens são button (teclado via Tab e Enter/Space).
  • Estado ativo comunica via aria-current="page".
  • Menu usa <nav aria-label="Menu"> para leitores de tela.
  • Botão colapsar usa aria-expanded e aria-controls ligados ao nav.
  • Ícones SVG vêm de ds-demo-icons.js (mesmos paths que icons-demo.html — Lucide no DS).
Preview
Anatomia do Componente (HTML)
  • nav.menu: container da sidebar (colapsável via classe collapsed).
  • button.menu-item: item de navegação.
  • span[data-ds-icon="…"]: ícone injetado por ds-demo-icons.js (mesmos nomes que em icons-demo.html).
  • button.menu-item.active: item selecionado (estilo e aria-current).
  • span.menu-item__label: texto (oculto quando .menu.collapsed).
Como Usar (HTML)
Inclua ds-demo-icons.js, use data-ds-icon="layout-grid|user|settings" em um span, e chame o injetor no load. menu.collapsed colapsa; menu-item.active marca o atual.
<script src="ds-demo-icons.js"></script>
<button type="button" id="menu-toggle-btn" aria-expanded="true" aria-controls="sidebar-menu" onclick="toggleMenu()">
  <span id="menu-toggle-text">Colapsar Menu</span>
</button>

<nav class="menu" id="sidebar-menu" aria-label="Menu">
  <button class="menu-item active" type="button" onclick="setActive(this)" aria-current="page">
    <span class="menu-item-icon" data-ds-icon="layout-grid" aria-hidden="true"></span>
    <span class="menu-item__label">Dashboard</span>
  </button>
</nav>

<script>
  function injectDsIcons() {
    document.querySelectorAll('[data-ds-icon]').forEach(function (el) {
      el.innerHTML = window.dsDemoIconSvg(el.getAttribute('data-ds-icon'));
    });
  }
  injectDsIcons();
</script>
Como Usar (React)
Passe items com iconName do DS (ex.: layout-grid) e renderize com os mesmos paths que ds-demo-icons.js.
{/* JSX */}
import { Menu } from '@/components/FinalComponents';

const menuItems = [
  { id: 'dashboard', label: 'Dashboard', iconName: 'layout-grid' },
  { id: 'users', label: 'Usuários', iconName: 'user' },
];

function Sidebar() {
  return (
    <Menu
      items={menuItems}
      collapsed={false}
      onItemClick={(item) => console.log(item.id)}
    />
  );
}
Props principais
items: {`{ id, label, iconName? }`}[] — iconName deve existir em DSDemoIconPaths / iconografia.
collapsed (opcional): boolean
onItemClick (opcional): (item) => void
