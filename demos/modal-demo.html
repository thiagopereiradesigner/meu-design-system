Modal

Diálogo para confirmar ações (overlay + fechamento por overlay, botão e Esc).

♿ Melhorias de Acessibilidade Aplicadas

  • Modal usa role="dialog" e aria-modal="true".
  • Fechamento por Esc e clique no overlay.
  • Foco inicial no botão de fechar ao abrir.
Preview

Confirmar Ação

Você tem certeza que deseja realizar esta ação?

Esta operação não pode ser desfeita.

Anatomia do Componente (HTML)
  • div.modal-overlay: overlay escurecida + fechamento ao clicar fora.
  • div.modal: container do diálogo (role/aria).
  • role="dialog" + aria-modal="true" para leitores de tela.
  • button.close-btn: fechar.
Como Usar (HTML)
Estruture overlay + modal e use as classes .show via JS.
{/* HTML + JS mínimo */}
<button type="button" id="modal-open-btn" onclick="openModal()">Abrir Modal</button>

<div class="modal-overlay" id="modal-overlay" onclick="closeModal()">
  <div class="modal" onclick="event.stopPropagation()" role="dialog" aria-modal="true" aria-labelledby="modal-title">
    <div class="modal-header">
      <h2 class="modal-title" id="modal-title">Confirmar Ação</h2>
      <button type="button" onclick="closeModal()">Fechar</button>
    </div>
    <div class="modal-content">...</div>
  </div>
</div>

<script>
  function openModal() {
    document.getElementById('modal-overlay').classList.add('show');
    document.body.style.overflow = 'hidden';
  }
  function closeModal() {
    document.getElementById('modal-overlay').classList.remove('show');
    document.body.style.overflow = '';
  }
  document.addEventListener('keydown', (e) => {
    if (e.key === 'Escape') closeModal();
  });
</script>
Como Usar (React)
Use isOpen e onClose; passe title e contenha ações em children.
{/* JSX */}
import { Modal } from '@/components/FinalComponents';
import React, { useState } from 'react';

function ConfirmDialog() {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <>
      <button type="button" onClick={() => setIsOpen(true)}>Abrir Modal</button>
      <Modal
        isOpen={isOpen}
        onClose={() => setIsOpen(false)}
        title="Confirmar Ação"
        size="medium"
        showCloseButton={true}
      >
        <p>Você tem certeza que deseja realizar esta ação?</p>
        <div style={{ display: 'flex', gap: 12, justifyContent: 'flex-end' }}>
          <button type="button">Confirmar</button>
          <button type="button">Cancelar</button>
        </div>
      </Modal>
    </>
  );
}
Props principais
isOpen: controla exibição.
onClose: callback ao fechar.
title: título do header.
size (opcional): 'small' | 'medium' | 'large'.
showCloseButton (opcional): exibe botão fechar.
