Checkbox

Caixa de seleção para várias opções independentes em formulários e filtros. Prefira <label> envolvendo o input para área de clique ampla e leitura correta por tecnologias assistivas.

♿ Nota Acessibilidade

  • Teclado: foco visível com :focus-visible e var(--ds-focus-ring); Estado marcado anunciado pelo leitor de tela.
  • Contraste: borda e estados usam tokens do tema (WCAG depende dos valores em ds-demo-base.css).
  • Toque: em telas estreitas, o rótulo + linha têm altura mínima ~44px (área confortável).
  • Erros: associe mensagem de erro ao grupo ou ao campo com aria-describedby quando aplicável.

Anatomia do Componente (HTML)

Estrutura usada nesta página:

  • label.checkbox-wrapper — região clicável; alinha ícone e texto.
  • input.checkbox-input — type="checkbox" nativo (teclado e AT).
  • span.checkbox-label — texto visível; mantenha próximo ao controle no DOM.
  • Estado marcado: estilo :checked + ::after com máscara SVG para o traço.

Como Usar (HTML)

Exemplo mínimo acessível (rótulo associado por aninhamento).

<label class="checkbox-wrapper">
  <input type="checkbox" class="checkbox-input" id="termos" />
  <span class="checkbox-label">Aceito os termos</span>
</label>

Como Usar (React)

Equivalente com o componente da pasta components/Forms/Checkbox.

import { useState } from 'react';
import { Checkbox } from '@/components/Forms/Checkbox/Checkbox';

export function Demo() {
  const [checked, setChecked] = useState(false);
  return (
    <Checkbox
      label="Aceito os termos e condições"
      checked={checked}
      onChange={setChecked}
    />
  );
}
