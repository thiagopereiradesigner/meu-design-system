✅ Checkbox

Controle de seleção múltipla, com suporte a estados marcado, desmarcado e desabilitado.

Basic

States

Anatomia do Componente (HTML)

Checkbox custom usando input nativo, para manter foco/teclado e leitura por leitores de tela.

  • label.checkbox-wrapper: aumenta a área clicável.
  • input.checkbox-input: input nativo (checkbox).
  • span.checkbox-label: texto acessível ao lado do checkbox.
  • Estados do demo: checked, disabled e variação de UI via CSS.

Como Usar (HTML)

Exemplo mínimo de estrutura para checkbox acessível. 

<label class="checkbox-wrapper">
  <input 
    type="checkbox" 
    class="checkbox-input" 
    checked />
  <span class="checkbox-label">Aceito os termos</span>
</label>

Como Usar (React)

JSX equivalente usando o componente do design system. 

{/* React */}
import React, { useState } from 'react';
import { Checkbox } from '@/components/Forms/Checkbox/Checkbox';

export default function Demo() {
  const [checked, setChecked] = useState(false);

  return (
    <Checkbox
      label="Aceito os termos e condições"
      checked={checked}
      onChange={setChecked}
    />
  );
}
Props principais
label, checked, onChange
disabled, indeterminate, error, helperText

♿ Nota Acessibilidade

  • Use <label> para aumentar a área clicável e facilitar navegação.
  • O input permanece nativo para suporte a teclado/AT.
  • Para erro, mantenha mensagem textual (error) e contraste.
