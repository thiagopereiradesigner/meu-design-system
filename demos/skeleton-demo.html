Skeleton

Placeholders de carregamento no padrão shadcn/ui: blocos neutros com animação pulse (avatar + linhas).

♿ Melhorias de Acessibilidade Aplicadas

  • Região de skeleton decorativa: use aria-hidden="true" nos blocos visuais.
  • Enquanto carrega conteúdo real, prefira role="status" + texto em .sr-only no container pai (ex.: “Carregando perfil”).
  • Animação de opacidade (pulse) — evite apenas spinner sem contexto quando o layout já indica “placeholder”.

Anatomia do Componente (HTML)
  • .skeleton-profile: flex horizontal, alinhamento vertical central, gap 16px.
  • .skeleton.skeleton-avatar: 48×48px, rounded-full (placeholder de avatar).
  • .skeleton-lines: coluna com gap 8px; linhas com altura 16px e larguras ~250px / ~200px, rounded-full (pílula).
  • .skeleton + @keyframes ds-skeleton-pulse: cor de fundo neutra e pulse de opacidade.
Como Usar (HTML)
Bloco perfil (shadcn-like) + região acessível no pai.
<div role="status" aria-busy="true">
  <span class="sr-only">Carregando perfil</span>
  <div class="skeleton-profile" aria-hidden="true">
    <div class="skeleton skeleton-avatar skeleton-rounded-full"></div>
    <div class="skeleton-lines">
      <div class="skeleton skeleton-line-title skeleton-rounded-full"></div>
      <div class="skeleton skeleton-line-sub skeleton-rounded-full"></div>
    </div>
  </div>
</div>
Como Usar (React)
Use Skeleton para peças soltas ou SkeletonProfileRow para o layout avatar + linhas.
import { Skeleton, SkeletonProfileRow } from '@/components/FinalComponents';

function LoadingProfile() {
  return (
    <div role="status" aria-busy="true">
      <span className="sr-only">Carregando perfil</span>
      <SkeletonProfileRow />
    </div>
  );
}

// Peça única (largura/altura/arredondamento)
<Skeleton width={120} height={14} rounded="full" />
Props — Skeleton
width / height: dimensões (ex.: '250px', 16).
rounded: 'none' | 'md' | 'full'
aria-hidden: padrão true (decorativo).
Props — SkeletonProfileRow
avatarSize, titleWidth, subtitleWidth, lineHeight, gap (opcionais).
