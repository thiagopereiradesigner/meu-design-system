💡 Exemplos Práticos de Uso

♿ Nota sobre Acessibilidade (WCAG)

  • Todos os botões possuem estados visuais claros (hover, focus, active)
  • Contrast ratio conforme WCAG AA (4.5:1 para texto normal)
  • Focus visível com outline de 3px para navegação por teclado
  • Estados desabilitados claramente diferenciados
  • Tamanho mínimo de toque: 40px (WCAG 2.5.5 - Level AAA)
  • Ícones com stroke visível em todos os estados
Anatomia do Componente (HTML)
  • button.btn: componente base.
  • .btn-primary|btn-secondary|btn-outline|btn-ghost: variação de estilo.
  • .btn-large|btn-medium: variação de tamanho (padding/font-size).
  • .disabled: estado desabilitado.
  • Ícone: span ou svg com classes como icon dentro do botão.
Como Usar (HTML)
{/* HTML */}
<button class="btn btn-primary btn-large">
  Criar novo
</button>

<button class="btn btn-outline btn-medium">
  Ver mais
  <svg class="icon" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">...</svg>
</button>
Como Usar (React)
JSX equivalente usando as mesmas classes do demo (copiável para IA/Dev). 
{/* React */}
Props principais (visuais)
variant (primary/secondary/outline/ghost)
size (large/medium)
disabled (true/false)
iconOnly (se você usar “só ícone”, garanta aria-label)

🎨 Variações de Ícones e Texto

1. Botão só com texto (sem ícones) 

<button class="btn btn-primary btn-large">Salvar</button>

2. Botão com ícone à esquerda 

<button class="btn btn-primary btn-large">
  <svg class="icon">...</svg>
  Criar novo
</button>

3. Botão com ícone à direita 

<button class="btn btn-outline btn-medium">
  Ver mais
  <svg class="icon">...</svg>
</button>

4. Botão icon-only (apenas ícone) ⭐ IMPORTANTE

⚠️ Acessibilidade: Sempre adicione aria-label e title em botões icon-only! 

<button 
  class="btn btn-primary btn-large" 
  style="width: 48px; padding: 12px;"
  aria-label="Adicionar novo item"
  title="Adicionar novo item"
>
  <svg class="icon">...</svg>
</button>
