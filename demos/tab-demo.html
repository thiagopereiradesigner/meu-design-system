Acessibilidade e contraste (WCAG)

  • ARIA: role="tablist", role="tab", aria-selected, aria-controls nos exemplos interativos; painéis com hidden quando inativos.
  • Teclado: setas alternam a aba focada e ativam o painel; Tab entra na lista.
  • Foco visível: apenas --ds-focus-ring (mesmo halo suave de inputs/botões; claro/escuro vêm do ds-demo-base.css).
  • Leitura: texto inativo em --ds-fg-muted e ativo em --ds-success sobre fundos de --ds-bg / --ds-surface; badges ativos usam --ds-on-inverse sobre --ds-success.
  • Ícones: decorativos com aria-hidden="true"; nome da aba sempre em texto.
Anatomia do Componente (HTML)
  • div.tab-group + tab-group--line ou tab-group--pill: envolve o tablist.
  • div[role="tablist"]: lista de abas; use aria-label para o propósito do grupo.
  • button[role="tab"]: cada aba; classes tab--line ou tab--pill, modificadores de tamanho (tab--md, tab--size-lg, etc.).
  • Ícone: SVG com classe tab__icon antes do texto (cor currentColor).
  • span.tab-badge: opcional; contadores com texto ≥ --ds-font-size-sm.
  • div[role="tabpanel"]: conteúdo; sincronize hidden e tab-panel--active com o script.
Como Usar (HTML)

Estrutura mínima com painel visível e demais painéis ocultos com hidden.

<div role="tablist" class="tab-group tab-group--line" aria-label="Seções">
  <button type="button" class="tab--line tab--active" role="tab" aria-selected="true" aria-controls="p1" id="t1">
    <svg class="tab__icon" aria-hidden="true">…</svg> Visão geral
  </button>
  <button type="button" class="tab--line" role="tab" aria-selected="false" aria-controls="p2" id="t2">Detalhes</button>
</div>
<div id="p1" role="tabpanel" aria-labelledby="t1" class="tab-panel tab-panel--active">…</div>
<div id="p2" role="tabpanel" aria-labelledby="t2" class="tab-panel" hidden>…</div>
Como Usar (React)

Componente Tabs com itens { id, label, icon? }; ícone opcional (nó React) é renderizado antes do rótulo.

import { Tabs } from '@/components/FinalComponents';

<Tabs
  tabs={[
    { id: 'a', label: 'Visão geral', icon: <IconHome aria-hidden /> },
    { id: 'b', label: 'Configurações' },
  ]}
  activeTab="a"
  onChange={(id) => {}}
  variant="underline"
/>
tabs
Array { id, label, icon? }; icon é opcional (ex.: SVG inline).
activeTab / onChange
Modo controlado da aba ativa.
variant
'underline' ou 'filled' (pílula).
