Tab

Componente Tab | Todas as variantes, tamanhos e estados

Tabs para alternar conteúdo por seção, com variantes (underline/filled), estados (active/disabled) e papéis ARIA.

♿ Melhorias de Acessibilidade Aplicadas

  • Use role="tablist", role="tab" e role="tabpanel" com aria-selected.
  • Para tabs desabilitadas, aplique disabled e use aria-disabled="true".
  • Associe cada aba com o painel via aria-controls e aria-labelledby.
Anatomia do Componente (HTML)
  • div.tab-group: wrapper do conjunto de abas (ex.: tab-group--line ou tab-group--pill).
  • div[role="tablist"]: container das abas.
  • button.tab--line / button.tab--pill: cada tab com role="tab".
  • aria-selected: indica qual aba está ativa.
  • aria-controls / aria-labelledby: relaciona tab e painel.
  • div.tab-panel: painel correspondente; tab-panel--active define a aba visível.
  • tab--disabled (e disabled): estado desabilitado da tab.
Como Usar (HTML)

Exemplo mínimo de uma tab group com 2 abas e 2 painéis (underline). 

<div role="tablist" class="tab-group tab-group--line" aria-label="Seções">
  <button
    class="tab--line tab--active"
    role="tab"
    aria-selected="true"
    aria-controls="panel-1"
    id="tab-1">
    Aba 1
  </button>
  <button
    class="tab--line"
    role="tab"
    aria-selected="false"
    aria-controls="panel-2"
    id="tab-2">
    Aba 2
  </button>
</div>

<div id="panel-1" role="tabpanel" aria-labelledby="tab-1" class="tab-panel tab-panel--active">
  <div class="tab-panel-card">Conteúdo da Aba 1</div>
</div>

<div id="panel-2" role="tabpanel" aria-labelledby="tab-2" class="tab-panel">
  <div class="tab-panel-card">Conteúdo da Aba 2</div>
</div>
Como Usar (React)

Snippet usando o componente Tabs do DS. 

import React from 'react';
import { Tabs } from '@/components/FinalComponents';

export function ExampleTabs() {
  return (
    <Tabs
      tabs={[
        { id: 'overview', label: 'Visão geral' },
        { id: 'settings', label: 'Configurações' },
      ]}
      activeTab="overview"
      onChange={(id) => console.log('tab changed', id)}
      variant="underline"
    />
  );
}
tabs
Array de abas: {`{ id, label }`}.
activeTab
Id da tab ativa (controlado). Se omitido, o componente usa estado interno.
onChange
Callback chamado com o tabId ao clicar em uma aba.
variant
'underline' ou 'filled'.

♿ Nota sobre Acessibilidade (WCAG)

  • Use role="tablist" no container e role="tab" em cada botão
  • Adicione aria-selected="true/false" para indicar o estado atual ao leitor de tela
  • Associe cada aba ao seu painel com aria-controls e aria-labelledby
  • Use aria-disabled="true" junto com disabled em abas desabilitadas
  • Forneça aria-label no tablist para descrever o grupo de abas
  • Suporte navegação por teclado: Tab para focar, ←/→ para navegar entre abas
  • O foco visível usa box-shadow de 2px na variante line e 3px na variante pill
  • Contrast ratio WCAG AA: texto ativo #04843B sobre branco = 5.5:1 ✓
