  • Use label textual associado ao textarea para contexto.

  • Para disabled, use o atributo disabled para impedir interação.

Anatomia do Componente (HTML)
  • div.textarea-wrapper: wrapper do textarea (label + campo + helper).
  • label.textarea-label: rótulo do campo.
  • textarea: controle nativo; use rows e maxlength.
  • textarea.error: estado de erro (borda/estilo).
  • textarea:disabled: estado desabilitado.
  • div.helper-text: mensagem de apoio.
  • div.char-count: contador (quando aplicável).

Como Usar (HTML)

Estrutura base para o uso de textarea com helper text e contador. 

<div class="textarea-wrapper">
  <label class="textarea-label">Comentário</label>
  <textarea 
    rows="4" 
    placeholder="Digite seu comentário..."></textarea>
  <div class="helper-text">Helper text opcional</div>
</div>
Como Usar (React)

Snippet React para controlar o valor do <textarea> e manter o mesmo HTML. 

import React, { useState } from 'react';

function TextareaField({
  label,
  placeholder = '',
  helperText = '',
  error = false,
  disabled = false,
  rows = 4,
  maxLength,
}) {
  const [value, setValue] = useState('');
  return (
    <div className="textarea-wrapper">
      <label className="textarea-label">{label}</label>
      <textarea
        rows={rows}
        placeholder={placeholder}
        className={error ? 'error' : undefined}
        disabled={disabled}
        maxLength={maxLength}
        value={value}
        onChange={(e) => setValue(e.target.value)}
      />
      {helperText ? <div className="helper-text">{helperText}</div> : null}
    </div>
  );
}
label
Texto exibido em label.textarea-label.
placeholder
Valor padrão do campo.
helperText
Conteúdo do bloco .helper-text.
error
Quando true, aplica classe error.
disabled
Quando true, aplica atributo disabled.
