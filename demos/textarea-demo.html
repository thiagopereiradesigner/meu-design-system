Textarea

Campo multilinhas com hover e foco alinhados ao Input/Select (--ds-success, --ds-focus-ring), estados de erro e texto auxiliar com contraste legível.

Estados normais, erro e desabilitado; helper e contador opcionais. Mobile-first com área de toque confortável.

Acessibilidade

  • label associado ao campo; aria-invalid quando houver erro.
  • Foco visível: --ds-focus-ring; texto auxiliar em --ds-fg-muted (contraste sobre --ds-bg).
  • disabled remove edição; cores em --ds-fg-muted / --ds-bg-subtle.

Textarea

Hover reforça borda com --ds-success (igual input/select); foco combina borda + halo do DS.

Anatomia do Componente (HTML)
  • div.textarea-wrapper: agrupa label, textarea e mensagens.
  • label.textarea-label + for apontando ao id do campo.
  • textarea: classes error opcional; disabled nativo.
  • div.helper-text / div.error-text / contador.
Como Usar (HTML)

Estrutura base com hover/focus via CSS do demo (tokens --ds-*).

<div class="textarea-wrapper">
  <label class="textarea-label" for="msg">Comentário</label>
  <textarea id="msg" rows="4" placeholder="…"></textarea>
  <div class="helper-text">Texto auxiliar</div>
</div>
Como Usar (React)

Componente Textarea em components/Forms/Textarea/Textarea.jsx com borda dinâmica e variáveis CSS para tema.

import { Textarea } from '@/components/Forms/Textarea/Textarea';

<Textarea
  label="Descrição"
  placeholder="…"
  helperText="Opcional"
  error=""
  rows={4}
/>
