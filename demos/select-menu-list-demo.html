Lista do select (menu)

O painel interno do Select (.select-dropdown) agrupa opções clicáveis (.select-option), opcionalmente com busca (.select-search). Cores, scrollbar e altura máxima usam tokens (var(--ds-*)) — novos temas podem sobrescrever essas variáveis. A página completa do campo está em Select.

♿ Nota Acessibilidade

  • Em implementações customizadas, considere role="listbox" no painel e role="option" em cada linha, com teclado (setas, Home/End, Escape) e foco visível.
  • Opções desabilitadas: use .disabled e aria-disabled="true"; não dispare seleção ao clicar.
  • Mantenha contraste de texto e de estados de hover/selecionado; teste zoom 200% e largura mínima ~320px.
  • Área tocável: linhas devem ser confortáveis em touch (padding vertical generoso já vem dos tokens do DS).
  • Listas longas: mantenha overflow-y: auto e altura máxima via var(--ds-select-dropdown-max-height); a barra de rolagem usa var(--ds-scrollbar-*) (Firefox scrollbar-color + WebKit).
  • Checkbox na linha: use <label class="select-option select-option--checkbox"> com input.select-option__checkbox; preserve foco visível e não feche o painel ao marcar sem querer (ex.: stopPropagation no container em JS se necessário).

Anatomia do Componente (HTML)

Partes do menu interno reutilizadas pelo Select custom e por outros painéis (drawer, modal).

  • div.select-dropdown: superfície rolável; max-height: var(--ds-select-dropdown-max-height); scrollbar via --ds-scrollbar-*. Classe .open controla visibilidade quando acoplada ao trigger.
  • div.select-search (opcional): wrapper do input de filtro.
  • div.select-option ou label.select-option: linha; modifique com .selected, .disabled, .select-option--row (ícone + texto), .select-option--checkbox (com input.select-option__checkbox).
  • svg.select-option__icon, span.select-option__label, svg.select-option__check: composição da linha com ícones.
  • div.select-dropdown__footer + button.select-dropdown__btn-primary: ação no rodapé do painel (tokens do DS).

  <!-- ... muitas linhas ... -->
</div>

  <input type="checkbox" class="select-option__checkbox" />
  <span class="select-option__label">Filtrar por arquivo</span>

  <button type="button" class="select-dropdown__btn-primary">Concluir</button>
</div>

  <svg class="select-option__icon" viewBox="0 0 24 24" aria-hidden="true">...</svg>
  <span class="select-option__label">PIX</span>
</div>

Como Usar (React)

Exemplo de dados para mapear linhas com ícone opcional e estado.

const options = [
  { id: 'pix', label: 'PIX', icon: <PixIcon className="select-option__icon" /> },
  { id: 'card', label: 'Cartão', disabled: true },
];

function SelectMenuList({ options, value, onSelect }) {
  return (
    <div className="select-dropdown open" role="listbox">
      {options.map((opt) => (
        <div
          key={opt.id}
          role="option"
          aria-selected={value === opt.id}
          aria-disabled={opt.disabled}
          className={
            'select-option select-option--row' +
            (value === opt.id ? ' selected' : '') +
            (opt.disabled ? ' disabled' : '')
          }
          onClick={() => !opt.disabled && onSelect(opt.id)}
        >
          {opt.icon}
          <span className="select-option__label">{opt.label}</span>
          {value === opt.id ? (
            <CheckIcon className="select-option__check" aria-hidden />
          ) : null}
        </div>
      ))}
    </div>
  );
}
Props / campos úteis
options[]: id, label, icon?, disabled?
value, onSelect(id) — espelha seleção atual.
Para checkboxes, mantenha estado em objeto ou Set; opcional onToggleFilter(key, checked).
Combine com o trigger do Select para abrir/fechar o painel.
