Card

O Card agrupa conteúdo relacionado: formulários de login, eventos, destaques com imagem ou variações estilo grid do portal. Use quando precisar de uma superfície legível, com hierarquia clara e ações no rodapé.

♿ Acessibilidade

  • Landmark: use article ou section com aria-labelledby apontando para o título.
  • Formulário: associe label + for / id; foco na ordem: título → campos → ações.
  • Imagem: alt descritivo em img.ds-card__media.
  • SSO: botão secundário visível (outline verde) mantém hierarquia com a ação primária “Entrar”.

Anatomia do Componente (HTML)

Partes usadas nesta página:

  • .ds-card + ds-card--outline | ds-card--elevated | ds-card--feature.
  • .ds-card__media — imagem no topo (ratio controlado por max-height).
  • .ds-card__body, .ds-card__header, .ds-card__title-row, .ds-card__title, .ds-card__action.
  • .ds-card__eyebrow + .ds-card__eyebrow--brand|--warm; .ds-card__description, .ds-card__content, .ds-card__divider, .ds-card__footer.
  • .badge: .solid-success, .soft-success, .solid-warning (texto --ds-warning-solid-fg), .soft-warning, .soft-error, .info, .neutral + .small|medium.
  • .btn + .btn-primary | .btn-secondary | .btn-outline | .btn-outline-success + .btn-block (SSO / secundário em verde).
  • .ds-showcase-card* — variante estilo grid do portal.

Como Usar (HTML)

Formulário com primária + SSO em outline verde.

<article class="ds-card ds-card--outline" aria-labelledby="login-title">
  <div class="ds-card__body">
    <div class="ds-card__header">
      <div class="ds-card__title-row">
        <h2 id="login-title" class="ds-card__title">Entrar</h2>
        <a href="/signup" class="ds-card__action">Criar conta</a>
      </div>
      <p class="ds-card__description">Use seu e-mail corporativo.</p>
    </div>
    <div class="ds-card__content">
      <!-- inputs -->
    </div>
    <hr class="ds-card__divider" role="separator" />
    <div class="ds-card__footer">
      <button type="button" class="btn btn-primary btn-block">Entrar</button>
      <button type="button" class="btn btn-outline-success btn-block">Continuar com SSO</button>
    </div>
  </div>
</article>

Como Usar (React)

Componentes em FinalComponents.jsx: Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter.

import {
  Card, CardHeader, CardTitle, CardDescription,
  CardContent, CardFooter,
} from './components/FinalComponents';

export function LoginCard() {
  return (
    <Card variant="outline" padding="md" style={{ maxWidth: 420 }}>
      <CardHeader action={<a href="/signup">Criar conta</a>}>
        <CardTitle id="login-title">Entrar na conta</CardTitle>
        <CardDescription>Digite seu e-mail.</CardDescription>
      </CardHeader>
      <CardContent>{/* inputs */}</CardContent>
      <CardFooter>
        <button type="button" className="btn btn-primary btn-block">Entrar</button>
        <button type="button" className="btn btn-outline-success btn-block">Continuar com SSO</button>
      </CardFooter>
    </Card>
  );
}

Props principais

  • variant: elevated | outline | ghost | feature
  • padding: none | sm | md | lg
  • as (Card): section | article | div
  • CardHeader: action opcional
  • No HTML/CSS do DS, use btn-outline-success para SSO alinhado à marca.
