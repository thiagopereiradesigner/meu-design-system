💻 Como usar (HTML)

1. Input com calendário (estrutura base)

<!-- Wrapper com posicionamento relativo -->
<div style="position: relative; width: fit-content;">

  <!-- Campo de entrada -->
  <div class="dp-field">
    <label class="dp-label" for="minha-data">
      Data <span class="dp-label-required">*</span>
    </label>
    <div class="dp-input-wrapper">
      <input
        type="text"
        class="dp-input"
        id="minha-data"
        placeholder="dd/mm/aaaa"
        readonly
      >
      <button class="dp-icon-btn" aria-label="Abrir calendário">
        <!-- ícone de calendário SVG -->
      </button>
    </div>
    <span class="dp-helper">Texto auxiliar</span>
  </div>

  <!-- Popover do calendário (JS controla visibilidade) -->
  <div class="dp-popover dp-popover--hidden" role="dialog" aria-label="Calendário">
    <!-- Header + grid de dias gerado via JS -->
  </div>

</div>

2. Date Range (dois campos)

<div class="dp-range-group">
  <div class="dp-field">
    <label class="dp-label">Data inicial</label>
    <div class="dp-input-wrapper">
      <input type="text" class="dp-input" placeholder="dd/mm/aaaa" readonly>
      <button class="dp-icon-btn" aria-label="Abrir calendário"><!-- SVG --></button>
    </div>
  </div>
  <div class="dp-range-separator" aria-hidden="true">
    <!-- seta SVG -->
  </div>
  <div class="dp-field">
    <label class="dp-label">Data final</label>
    <div class="dp-input-wrapper">
      <input type="text" class="dp-input" placeholder="dd/mm/aaaa" readonly>
      <button class="dp-icon-btn" aria-label="Abrir calendário"><!-- SVG --></button>
    </div>
  </div>
</div>

♿ Nota sobre Acessibilidade (WCAG)

  • Use role="dialog" e aria-label="Calendário" no popover para leitores de tela
  • O botão de ícone deve ter aria-label="Abrir calendário" e aria-expanded atualizado via JS
  • Dias desabilitados devem ter aria-disabled="true" além do atributo disabled
  • Dias selecionados recebem aria-pressed="true" ou aria-selected="true"
  • Suporte a teclado: Enter/Space para selecionar, ←→↑↓ para navegar, Esc para fechar
  • O campo de entrada usa readonly para evitar digitação livre — use máscara ou apenas o calendário
  • Focus visível: box-shadow: 0 0 0 3px rgba(4, 132, 59, 0.1) no input, borda verde no popover
  • Contrast ratio WCAG AA: texto selecionado branco sobre #04843B = 5.5:1 ✓
