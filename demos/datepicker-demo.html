Date Picker

Componente Date Picker | Todos os estados, variantes e interações

Date Picker com popover de calendário, suporte a teclado e variantes (single date e date range).

♿ Melhorias de Acessibilidade Aplicadas

  • Popover do calendário como role="dialog" e com aria-label.
  • Trigger (botão/ícone) com aria-label e estado atualizado (aria-expanded).
  • Tratamento de disabled com atributos compatíveis e mensagens textuais.
  • Navegação por teclado (setas, Enter/Space, Esc).

Anatomia do Componente (HTML)
  • div.dp-field: wrapper do campo (label + input + helper).
  • label.dp-label e span.dp-label-required: rótulo e obrigatoriedade.
  • div.dp-input-wrapper: wrapper relativo que posiciona o botão/ícone.
  • input.dp-input: campo de data (geralmente readonly).
  • button.dp-icon-btn: botão ícone que abre o popover (com aria-label).
  • div.dp-popover: popover do calendário (com role="dialog").
  • button.dp-day: células dos dias, incluindo modificadores como dp-day--selected.
  • Range: div.dp-range-group, e classes de dias como dp-day--range-start / dp-day--range-end.

Como Usar (HTML)

1. Input com calendário (estrutura base)

<!-- Wrapper com posicionamento relativo -->
<div style="position: relative; width: fit-content;">

  <!-- Campo de entrada -->
  <div class="dp-field">
    <label class="dp-label" for="minha-data">
      Data <span class="dp-label-required">*</span>
    </label>
    <div class="dp-input-wrapper">
      <input
        type="text"
        class="dp-input"
        id="minha-data"
        placeholder="dd/mm/aaaa"
        readonly
      >
      <button class="dp-icon-btn" aria-label="Abrir calendário">
        <!-- ícone de calendário SVG -->
      </button>
    </div>
    <span class="dp-helper">Texto auxiliar</span>
  </div>

  <!-- Popover do calendário (JS controla visibilidade) -->
  <div class="dp-popover dp-popover--hidden" role="dialog" aria-label="Calendário">
    <!-- Header + grid de dias gerado via JS -->
  </div>

</div>

Como Usar (React)

Snippet React mantendo as mesmas classes do HTML: trigger com aria-expanded e popover como role="dialog". 

import React, { useState } from 'react';

function DatePickerField({
  label,
  placeholder = 'dd/mm/aaaa',
  value,
  onChange,
  disabledBefore,
}) {
  const [open, setOpen] = useState(false);

  return (
    <div className="dp-field">
      <label className="dp-label">{label}</label>
      <div className="dp-input-wrapper">
        <input
          className="dp-input"
          placeholder={placeholder}
          readOnly
          value={value || ''}
        />
        <button
          className="dp-icon-btn"
          aria-label="Abrir calendário"
          aria-expanded={open}
          aria-haspopup="dialog"
          onClick={() => setOpen((v) => !v)}
        >
          <!-- SVG do calendário -->
        </button>
      </div>

      {open ? (
        <div className="dp-popover" role="dialog" aria-label="Calendário">
          <!-- render do calendário (dias) e handlers de seleção -->
        </div>
      ) : null}
    </div>
  );
}
label
Rótulo do campo (use dp-label).
value
String da data exibida no input (read-only).
onChange
Callback chamado quando o usuário seleciona uma data.
disabledBefore
Data limite para desabilitar dias anteriores.
