Divider

Traço horizontal/vertical com cores e espessuras configuráveis.

♿ Melhorias de Acessibilidade Aplicadas

  • Separador semântico: usa role="separator".
  • Orientação: aria-orientation acompanha horizontal/vertical.

Anatomia do Componente (HTML)
  • div.divider: separador base (horizontal).
  • div.divider.thick: variação de espessura.
  • div.divider.vertical: separador vertical.
  • role="separator" + aria-orientation: melhora entendimento para leitores de tela.
A orientação (horizontal/vertical) deve ser consistente com o layout.
Como Usar (HTML)
Estruture o divider como uma única linha div, ajustando classes e atributos de acessibilidade.
{/* HTML */}
<div class="divider" role="separator" aria-orientation="horizontal"></div>
<div class="divider thick" role="separator" aria-orientation="horizontal"></div>

<div style="display:flex;align-items:center;">
  <div>Coluna A</div>
  <div class="divider vertical" role="separator" aria-orientation="vertical"></div>
  <div>Coluna B</div>
</div>
Como Usar (React)
Use `Divider` para separar seções/estados e melhorar leitura. Para acessibilidade, ele usa `role="separator"`. 
        {/* JSX */}
import { Divider } from '@/components/ui/Divider';
Props principais
orientation: horizontal | vertical
thickness: espessura (number)
color: cor (string/var)
margin: espaçamento (string/number)
