💻 Como usar (HTML)

Exemplo mínimo de estrutura para o componente de Select básico. 

<div class="field">
  <div class="field-label">
    <span class="field-label-text">Estado</span>
    <span class="field-sublabel">obrigatório</span>
  </div>
  <div class="select-container">
    <div class="field-input-container" onclick="toggleDropdown('estado')">
      <span class="select-text" id="estado-text">Selecione um estado</span>
      <span>▾</span>
    </div>
    <div class="select-dropdown" id="estado-dropdown">
      <div class="select-option" onclick="selectOption('estado', 'São Paulo', this)">
        <span>São Paulo</span>
      </div>
      <!-- ... -->
    </div>
  </div>
</div>
label / sublabel
Controle de textos acima do campo, seguindo o padrão de tipografia do design system.
estado selecionado
Controlado via elemento com id [id]-text e classe .has-value para mudar a cor do texto.
estados
Use .error na .field-input-container e em .field-helper para erro, e .disabled para estado desabilitado.
