Select

Componente Select | Estados, busca e código de uso

Dropdown de seleção única com estados (normal, erro e disabled), com opção de busca no conteúdo. Estilos partilhados em ds-demo-base.css; comportamento em ds-demo-select.js (reutilizado no Drawer e noutras demos).

♿ Melhorias de Acessibilidade Aplicadas

  • Garanta que a área clicável do trigger possua rótulo textual (via field-label).
  • Para erro, aplique .error no container e use helper text como mensagem única de feedback.
  • Para disabled, combine .disabled no trigger e bloqueie interações via JS.

Anatomia do Componente (HTML)
  • div.field: wrapper do componente.
  • div.field-label: container da label; use span.field-label-text e opcional span.field-sublabel.
  • div.select-container: wrapper do trigger + dropdown.
  • div.field-input-container: área clicável do trigger (aplique .error ou .disabled aqui).
  • span.select-text: texto exibido no trigger (use .has-value quando tiver valor selecionado).
  • div.select-dropdown: container das opções (aparece/oculta via .open).
  • div.select-option: cada opção do dropdown; opções desabilitadas devem ter classe disabled.
  • div.field-helper: mensagem abaixo do campo; use .error para erro.

Como Usar (HTML)

Exemplo mínimo de estrutura para o componente de Select básico. 

<div class="field">
  <div class="field-label">
    <span class="field-label-text">Estado</span>
    <span class="field-sublabel">obrigatório</span>
  </div>
  <div class="select-container">
    <div class="field-input-container" onclick="toggleDropdown('estado')">
      <span class="select-text" id="estado-text">Selecione um estado</span>
      <span>▾</span>
    </div>
    <div class="select-dropdown" id="estado-dropdown">
      <div class="select-option" onclick="selectOption('estado', 'São Paulo', this)">
        <span>São Paulo</span>
      </div>
      <!-- ... -->
    </div>
  </div>
</div>
label / sublabel
Controle de textos acima do campo, seguindo o padrão de tipografia do design system.
estado selecionado
Controlado via elemento com id [id]-text e classe .has-value para mudar a cor do texto.
estados
Use .error na .field-input-container e em .field-helper para erro, e .disabled para estado desabilitado.
Como Usar (React)

Snippet React que espelha o HTML: trigger controlado + dropdown com lista de opções. 

import React, { useMemo, useState } from 'react';

function SelectField({
  label,
  sublabel,
  options,
  placeholder = 'Selecione...',
  helperText = 'Selecione uma opção',
  error = false,
  disabled = false,
}) {
  const [open, setOpen] = useState(false);
  const [value, setValue] = useState('');
  const text = value || placeholder;

  const isError = Boolean(error);
  const selected = useMemo(
    () => options.find((o) => o.value === value) || null,
    [options, value]
  );

  return (
    <div className="field">
      <div className="field-label">
        <span className="field-label-text">{label}</span>
        {sublabel ? <span className="field-sublabel">{sublabel}</span> : null}
      </div>

      <div className="select-container">
        <div
          className={"field-input-container" + (isError ? " error" : "") + (disabled ? " disabled" : "")}
          onClick={() => !disabled && setOpen((v) => !v)}
          role="button"
          aria-expanded={open}
        >
          <span className={"select-text" + (value ? " has-value" : "")}>{text}</span>
          <span>▾</span>
        </div>

        <div className={"select-dropdown" + (open ? " open" : "")}>
          {options.map((opt) => (
            <div
              key={opt.value}
              className={"select-option" + (opt.disabled ? " disabled" : "")}
              onClick={() => {
                if (opt.disabled) return;
                setValue(opt.value);
                setOpen(false);
              }}
            >
              <span>{opt.label}</span>
            </div>
          ))}
        </div>
      </div>

      <div className={"field-helper" + (isError ? " error" : "")}>
        <span>{helperText}</span>
      </div>
    </div>
  );
}
options
Array com {`{ value, label, disabled? }`}.
error
Quando true, aplica classes error.
disabled
Quando true, bloqueia abertura e aplica classe disabled.
