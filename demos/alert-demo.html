Alert

Alert inline (variação do Toast)

Ícone por tipo e opção de texto. Também inclui botão de fechar (dismiss).

♿ Melhorias de Acessibilidade Aplicadas

  • Semântica: `role="alert"` no componente React.
  • Controle visual: alternativa `showText={false}` mantém título sem reduzir entendimento.
  • Fechamento: botão de dismiss com aria-label.
Anatomia do Componente (HTML)
  • div.alert: wrapper do alerta (modificadores active/light e success|error|warning|info).
  • role="alert": anuncia conteúdo para leitores de tela (quando renderizado no fluxo).
  • svg: ícone do tipo (success/error/warning/info).
  • div.alert-text: agrupador do texto.
  • div.alert-title: sempre exibido quando existe título.
  • div.alert-message: body/mensagem (pode ser omitido para “ícone sem body”).
  • button.alert-close: botão de fechar com aria-label.
Como Usar (HTML)
Para “ícone sem body”, não renderize .alert-message (mas mantenha .alert-title). 
{/* HTML */}
<div class="alert active success" role="alert">
  <svg ...></svg>
  <div class="alert-text">
    <div class="alert-title">Sucesso</div>
    <div class="alert-message">Operação concluída com sucesso.</div>
  </div>
  <button class="alert-close" aria-label="Fechar alerta" onclick="...">...</button>
</div>

<div class="alert active warning" role="alert">
  <svg ...></svg>
  <div class="alert-text">
    <div class="alert-title">Atenção</div>
    <!-- sem .alert-message -->
  </div>
  <button class="alert-close" aria-label="Fechar alerta" onclick="...">...</button>
</div>

Como Usar (React)
Alert inline é um componente “dentro do fluxo” (não fixa na tela). Ele pode exibir message e/ou apenas title (quando showText={false}). 
        {/* JSX */}
import { Alert } from '@/components/ui/Alert';
Props principais
type: success | error | warning | info
variant: active | light
showText: controla exibição do body (message)
maxWidth: evita alerta “esticar a tela” (default: 720px)
dismissible/onDismiss: controle do fechar
